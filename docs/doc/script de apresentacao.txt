Todas essas ferramentas foram escolhidas pela familiaridade do time com elas, 
pela solidez do framework no mercado, fasta documentação e comunidade bem ativa.


SCRUM e XP 

	Usamos boas práticas do scrum e xp para nos auxiliar na gestão e desenvolvimento do sistema. 
	Por ter conceitos dinamicos de gestão e desenvolvimento.

UML

	Usamos a UML para a modelagem do sistema e o JUDE como ferramenta case para isso. Por se 
	adquar melhor no desenvolvimento orientado a objeto.

Java

	O java como linguagem de programação, para denvolvimento do sistema. Pela maior familiariedade 
	do time de desenvolvimento, solidez de mercado, fasta documentação e comunidade ativa e colaborativa.

	Aqui também utilizamos alguns frameworks, para nos auxiliar e facilitar no desenvolvimento da aplicação.
	
	JSF para componetes de tela, usamos ele pela familiaridade do time com o framework, 
	e pela facilidade do uso do ajax com o java.

	Hibernate para persistencia de dados, toda aparte de iteração da aplicação do sistema 
	com o banco de dados e feito com ele. 
	Spring para aparte de controle MVC e Injeção de Dependêcia. Escolhemos pela familiaridade do time com o framework, 
	pela solidez do framework no mercado, pela fasta documentação e comunicadade em volta desse framework.

MySQL

	Como banco de dados. Escolhemos pelo facilidade de uso e familiaridade do time, com o banco. 
	E por se adquar bem com sistemas web.

HTML, CSS e Javascript
	Para custumizar e modelar a parte visual.


Tomcat
	Como servidor de aplicação. Escolhemos e pela ficilidade de admistração e pela familiaridade do time
	com o servidor.

Eclipse
	Como IDE de desenvolvimento. Escolhemos também pela familiedade do time com a ferramenta.

Datascraping
	Como a ANP, não fornece de uma forma amigavel os dados, como XML ou JSON. 
	
	Usamos uma técnica conhecida como datascraping, uma técnica que consiste na raspagem dos dados do site. 
	Raspamos o HTML e obtemos as informações que desejamos. 
	
	Para isso, usamos uma site conhecido como  ScrapperWiki, é um site onde as pessoas podem fazer scripts de 
	para scrapping em python.
